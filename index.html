Ebayfactory

WCF Service For Ebay MVC Model

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the WCF EbayFactory Home Page

EbayFactory is the back end for an MVC page that displays and allows the search of Ebay listings. The WCF service incorporates the Ebay API through a WCF extension. It also takes full advantage of LINQ objects as well as LINQ queries in order to utilize the MsSql backend database.
